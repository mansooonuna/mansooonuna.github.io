* Jekyll라는 정적 웹사이트 생성기를 사용
  
  - data : 블로그 이용에 필요한 여러 버튼 문구 등을 관리하는 파일 들이 있는 폴더
  - includes : 블로그 구성에 필요한 컴포넌트를 모아둔 폴더
  - layouts : 구조를 커스터마이징하는 폴더
  - pages : 페이지를 관리하는 폴더
  - posts : 작성한 게시물, 마크다운 파일
  - sass : 디자인을 관리하는 코드
  - assets : 구성에 필요한 자료
